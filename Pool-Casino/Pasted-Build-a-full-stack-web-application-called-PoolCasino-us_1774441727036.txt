Build a full-stack web application called **"PoolCasino"** using Node.js, Express, SQLite, and a clean HTML/CSS/JavaScript frontend.

This is a **fake-money casino simulator with a shared global pool economy**.

---

# 🧱 CORE FEATURES

## 1. Authentication System

Users can:

* Sign up with:

  * username
  * password
  * optional email
* Login/logout with sessions

Store in database:

* id
* username
* password_hash
* email (nullable)
* balance (default: 10000.00)
* created_at

Passwords must be securely hashed.

---

## 2. Global Pool Economy

Create a **single global pool**:

* Stores total money available in the system
* Includes:

  * house reserve
  * player losses

### Rules:

* When a player loses → pool increases
* When a player wins → pool decreases

Display publicly:

* total pool value
* biggest win
* biggest bet
* recent large bets

---

## 3. Betting System (Dynamic Probability)

Implement a scalable betting system:

### Requirements:

* Minimum bet: $0.01
* Default max bet: $10,000
* Max bet dynamically scales:
  max_bet = pool * 0.1

### Probability behavior:

* Lower bets → higher chance to win
* Higher bets → lower chance to win

Example scaling:

* $0.01 → ~99.99% win chance
* $10 → ~80%
* $100 → ~50%
* $1,000 → ~10%
* $10,000 → ~0.01%

Probability must adjust dynamically based on bet size relative to pool.
