Extend the existing Node.js + Express + SQLite fake-money casino app by adding **10 additional games** using a modular game system.

All games must:

* Use server-side logic only (no client trust)
* Integrate with user balances and global pool
* Log every bet
* Follow shared payout + probability rules
* Be easily extendable for future games

# 🧱 CORE GAME ENGINE (IMPORTANT)

Create a reusable engine:

function processBet(userId, gameType, betAmount, gameData)

Responsibilities:

* Validate bet (min $0.01, max based on pool)
* Ensure user has enough balance
* Calculate outcome
* Apply house edge
* Update:

  * user balance
  * global pool
* Log bet in DB
* Return structured result

# 1. 🎯 High-Low Card

## Gameplay:

* Draw a card (1–13)
* Player guesses if next card is higher or lower

## Logic:

* Random card generation

## Payout:

* correct: 1.8x
* incorrect: lose

# 2. 🎲 Double Dice

## Gameplay:

* Roll 2 dice
* Bet on:

## Payout:

* exact total: up to 10x
* even/odd: 1.9x

# 3. 💣 Risk Ladder

## Gameplay:

* random fail point

## Player:

# 4. 🎴 War (card game)

## Gameplay:

* Player vs dealer
* each draws a card

## Outcome:

* higher card wins

## Payout:

* win: 2x

# 5. 🎯 Target Multiplier

## Gameplay:

* player selects multiplier (1.5x–10x)

## Win:

# 6. 🧊 Ice Break (risk grid)

## Gameplay:

# 7. 🎡 Advanced Wheel

# 8. 🎲 Number Range Bet

# 9. 🔺 Pyramid Pick
